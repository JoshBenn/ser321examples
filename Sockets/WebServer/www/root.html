You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns two given values multiplied together
    • Parameters: num1=int, num2=int
    • Example: /multiply?num1=5&num2=10
  • /encrypt -- encrypts a given string with a given key
    • Parameters: data=string, key=string
    • Example: /encrypt?data=hello&key=world
  • /decrypt -- decrypts a given string with a given key
    • Parameters: data=string, key=string
    • Example: /decrypt?data=nkrru&key=world
